Forma Canonică
evidențiază vârful
f(x) = a(x − m)² + n
unde V(m, n) este vârful parabolei
Abscisa vârfului
m = −b2a
Ordonata vârfului
n = −Δ4a

Completarea pătratului

ax²+bx+c
= a(x² + bax) + c
= a(x + b2a)² − b²4a + c
= a(x + b2a)² − Δ4a
Axa de simetrie:  x = m = −b/(2a)
3
Discriminantul Δ
natura rădăcinilor
Δ = b² − 4ac
Δ > 0
Două rădăcini reale

x₁ ≠ x₂, ambele reale
Parabola taie Ox în 2 puncte

Δ = 0
Rădăcină dublă

x₁ = x₂ = −b/(2a)
Parabola tangentă la Ox

Δ < 0
Nicio rădăcină reală

Rădăcini complexe
Parabola nu taie Ox

Formula discriminantului vine din:
x₁,₂ = (−b ± √Δ) / 2a  →  sub radical trebuie Δ ≥ 0
Legătura cu vârful:
n = −Δ/(4a)  →  semnul lui Δ determină dacă vârful e sub, pe sau deasupra axei Ox

soluțiile lui f(x) = 0

Relațiile lui Viète
rădăcini ↔ coeficienți
Suma rădăcinilor
x₁ + x₂ = −ba
Produsul rădăcinilor
x₁ · x₂ = ca

Reconstituirea ecuației

x² − (x₁+x₂)·x + x₁·x₂ = 0

Cazuri speciale

  • b = 0  →  x₁ = −x₂ (simetrice față de O)
  • c = 0  →  una din rădăcini este 0
  • a = c  →  x₁ · x₂ = 1 (rădăcini inverse)
  • a+b+c = 0  →  x₁ = 1 este rădăcină
