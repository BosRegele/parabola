Parabola

Demonstrație

Completarea pătratului

ax²+bx+c
= a(x² + bax) + c

Cheia întregii funcții

Discriminantul Δ

Definiție
Δ = b² − 4ac
numărul Δ se numește discriminantul ecuației ax² + bx + c = 0
Δ > 0
Două rădăcini reale

x₁ ≠ x₂, ambele reale.
Parabola taie Ox în 2 puncte

Δ = 0
Rădăcină dublă

x₁ = x₂ = −b/(2a).
Parabola tangentă la Ox

Δ < 0
Nicio rădăcină reală

Rădăcini complexe.
Parabola nu taie Ox

Δ apare peste tot: în formula rădăcinilor x₁,₂ = (−b ± √Δ)/2a și în ordonata vârfului n = −Δ/(4a). Semnul lui decide totul.
Unde f(x) = 0

Reconstituirea ecuației

x² − (x₁+x₂)·x + x₁·x₂ = 0

Cazuri speciale

  • b = 0 → x₁ = −x₂ (simetrice față de O)
  • c = 0 → una din rădăcini este 0
  • a = c → x₁ · x₂ = 1 (rădăcini inverse)
  • a+b+c = 0 → x₁ = 1 este rădăcină

Identitatea geometrică

Focarul și directoarea

Definiție
PF = dist(P, d)
Fie un punct F și o dreaptă d care nu trece prin el. Parabola de focar F și directoare d este locul geometric (= mulțimea) punctelor P egal depărtate de F și de d. Notăm cu p distanța de la vârf la focar.
Demonstrație · vârful în origine
P(x, y), F(0, p), d: y = −p. Egalăm distanțele, ridicăm la pătrat:
x² + (y−p)² = (y+p)²  ⇒  x² = 4py
⇒ y = 14p·x² — deci a = 14p, adică p = 14a ∎
Teoremă
Graficul oricărei funcții f(x) = ax²+bx+c este o astfel de parabolă, cu p = 1/(4a), focarul F(m, n+p) și directoarea y = n−p — rezultă prin translația vârfului (distanțele nu se schimbă).

Două drumuri, aceeași dreaptă

Tangenta, pe două căi

Definiție
T: y = f(x₀) + f′(x₀)(x − x₀)
Tangenta în punctul T(x₀, f(x₀)) = poziția limită a secantei prin T; panta ei este derivata f′(x₀) = 2ax₀ + b (a XI-a). Normala = perpendiculara pe tangentă în T.
Teoremă · criteriul Δ = 0 (a IX-a)
Dreapta y = sx + ℓ (s = panta, ℓ = ordonata la origine) este tangentă parabolei ⟺ intersecția lor are soluție dublă:
ax² + (b−s)x + (c−ℓ) = 0 cu Δ = 0 ⟺ ℓ = c − (s−b)²4a
Demonstrație · cele două căi coincid
Cu s = 2ax₀+b, ambele dau ℓ = c − ax₀² — aceeași dreaptă ∎. Butonul „înfășurătoare” desenează numai tangentele: parabola apare singură, ca silueta lor.

Oglinda perfectă

Reflexia în focar

Teoremă · proprietatea optică
raze ∥ axă  ⟶  focar
Orice rază paralelă cu axa de simetrie, reflectată de parabolă după legea reflexiei (unghiul de incidență = unghiul de reflexie, față de tangentă), trece prin focarul F.
Demonstrație
P(t, at²), Q = proiecția lui P pe directoare ⇒ PF = PQ
Mediatoarea lui [FQ] (perpendiculara pe mijlocul lui): panta t2p = 2at = f′(t) și trece prin P ⇒ e tangenta
Triunghi isoscel ⇒ unghiuri egale ⇒ raza verticală se reflectă spre F ∎
Observație · aplicații
Antene satelit, faruri auto, telescoape, cuptoare solare — toate sunt parabole care adună sau emit razele prin focar.

Cercul osculator

Definiție
cerc(P₋ₕ, P₀, P₊ₕ) h→0⟶ cerc osculator
Cercul osculator într-un punct al curbei = poziția limită a cercului prin 3 puncte ale curbei, când ele se strâng spre punctul dat; raza lui R = raza de curbură. Evoluta = locul geometric al centrelor acestor cercuri.
Teoremă + demonstrație · în vârf
Cercul prin V(0,0) și punctele (±h, ah²) are raza:
h² + a²h⁴ = 2ah²r ⇒ r = 1 + a²h²2a ⟶ R = 12a când h→0 ∎
Observație
În vârf R = 1/(2a) = 2p — exact dublul distanței focale. Parabola e cea mai curbată în vârf și tot mai „dreaptă” spre infinit.

Arhimede și 4/3

Teorema lui Arhimede
Asegment = 43 · Atriunghi
Segment parabolic = regiunea dintre o coardă și arcul de parabolă. Triunghiul înscris: baza = coarda, al treilea vârf = punctul unde tangenta e paralelă cu coarda. („Cvadratura parabolei”, sec. III î.Hr.)

Pe y = x², coarda de la p la q are panta p+q; tangenta are panta 2x ⇒ paralele la x = (p+q)/2. Consecință: fiecare generație de triunghiuri are ¼ din aria precedentei.

Conica generală și matricea ei

Definiție
Ax² + Bxy + Cy² + Dx + Ey + F = 0
Conică = mulțimea punctelor (x, y) din plan care verifică o ecuație de gradul al II-lea ca mai sus (A, B, C nu toate nule)
scrierea matricială · XT·M·X = 0
[x y 1] AB/2D/2B/2CE/2D/2E/2F [x y 1]T
Teoremă · clasificarea (o admitem)
B²−4AC < 0 elipsă · = 0 parabolă · > 0 hiperbolă. Intuiția: Ax²+Bxy+Cy² e pătrat perfect exact când B²−4AC = 0 — același Δ din clasa a IX-a. Parabola e frontiera dintre elipsă și hiperbolă.
De ce „conice”? Sunt secțiunile unui con: planul paralel cu o generatoare (o dreaptă de pe suprafața conului) decupează parabola; mai „culcat” — elipsa; mai „vertical” (taie ambele pânze) — hiperbola.

Un fapt contraintuitiv

Toate parabolele sunt asemenea

Definiție
(x, y) ⟼ (λx, λy)
Asemănarea de raport λ > 0 = transformarea care înmulțește toate distanțele cu λ (un „zoom” uniform al planului). Două figuri sunt asemenea dacă una e imaginea celeilalte printr-o asemănare.
Teoremă + demonstrație
Oricare două parabole sunt asemenea. Fie y = ax² și X = λx, Y = λy:
Y = λ·ax² = λa(X/λ)² = (a/λ)·X²
Cu λ = a/a′, parabola „a” devine parabola „a′” ∎
Observație
Ca cercurile: nu există parabole „mai late” sau „mai înguste” — doar văzute mai de aproape sau mai de departe. La elipse nu funcționează: raportul semiaxelor rezistă la zoom.
