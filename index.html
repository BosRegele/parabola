Descoperă
1

Forma Generală

Reprezentarea standard a funcției pătratice

f(x) = ax² + bx + c

Condiții obligatorii

  • a, b, c ∈ ℝ — coeficienți reali
  • a ≠ 0 — condiție esențială (altfel funcția devine liniară)
  • Gradul funcției este 2
  • Domeniu: D = ℝ

Semnificația coeficienților

Coef.RolEfect grafic
aCoeficient principalDeschidere ramuri (sus/jos)
bCoeficient liniarDeplasare orizontală vârf
cTermen liberIntersecție Oy: f(0) = c
Codomeniu: dacă a > 0 → [n, +∞)  |  dacă a < 0 → (−∞, n]
2

Forma Canonică

Forma care evidențiază direct vârful parabolei

f(x) = a(x − m)² + n
unde V(m, n) este vârful parabolei
Abscisa vârfului
m = −b2a
Ordonata vârfului
n = −Δ4a

Completarea pătratului

f(x) = ax² + bx + c
= a(x² + bax) + c
= a(x + b2a)² − b²4a + c
= a(x + b2a)² − Δ4a

Avantaje

  • Identificare imediată a vârfului V(m, n)
  • Determinarea min / max al funcției
  • Axa de simetrie: x = m
3

Discriminantul Δ

Determină natura rădăcinilor ecuației pătratice

Δ = b² − 4ac
Δ > 0
Două rădăcini reale distincte

x₁ ≠ x₂, ambele reale
Parabola intersectează Ox în 2 puncte

Δ = 0
O rădăcină reală dublă

x₁ = x₂ = −b/(2a)
Parabola este tangentă la Ox

Δ < 0
Nicio rădăcină reală

Rădăcini complex conjugate
Parabola nu intersectează Ox

4

Rădăcinile Ecuației

Soluțiile ecuației f(x) = 0, adică ax² + bx + c = 0

x1,2 = −b ± √Δ2a
Valabilă pentru Δ ≥ 0
Δ > 0:  x1 = −b − √Δ2a  ,  x2 = −b + √Δ2a
Δ = 0:  x1 = x2 = −b2a
Δ < 0:  ∄ soluții reale

Proprietăți ale rădăcinilor

ProprietateFormula
Sumăx₁ + x₂ = −b/a
Produsx₁ · x₂ = c/a
Diferență|x₁ − x₂| = √Δ / |a|
Factorizaref(x) = a(x − x₁)(x − x₂)
Forma produsului:  f(x) = a(x − x₁)(x − x₂),  dacă Δ ≥ 0
5

Relațiile lui Viète

Legătura directă dintre rădăcini și coeficienți

Suma rădăcinilor
x₁ + x₂ = −ba
Produsul rădăcinilor
x₁ · x₂ = ca

Aplicații practice

Reconstituirea ecuației din rădăcini cunoscute:

x² − (x₁+x₂)x + x₁·x₂ = 0
x² + bax + ca = 0

Cazuri speciale:

  • Dacă b = 0: x₁ = −x₂ (simetrice față de O)
  • Dacă c = 0: una din rădăcini este 0
  • Dacă a = c: x₁ · x₂ = 1 (rădăcini inverse)
  • Dacă a + b + c = 0: x₁ = 1 este rădăcină

Semnul Funcției

Când este f(x) pozitivă, negativă sau zero

a > 0

Ramuri în sus ↑

Δ > 0 — x₁ < x₂:

f(x) > 0 pentru x ∈ (−∞, x₁) ∪ (x₂, +∞)
f(x) < 0 pentru x ∈ (x₁, x₂)
f(x) = 0 pentru x = x₁ sau x = x₂

Δ = 0 — x₁ = x₂ = m:

f(x) ≥ 0 pentru orice x ∈ ℝ
f(x) = 0 doar pentru x = m

Δ < 0:

f(x) > 0 pentru orice x ∈ ℝ
a < 0

Ramuri în jos ↓

Δ > 0 — x₁ < x₂:

f(x) < 0 pentru x ∈ (−∞, x₁) ∪ (x₂, +∞)
f(x) > 0 pentru x ∈ (x₁, x₂)
f(x) = 0 pentru x = x₁ sau x = x₂

Δ = 0 — x₁ = x₂ = m:

f(x) ≤ 0 pentru orice x ∈ ℝ
f(x) = 0 doar pentru x = m

Δ < 0:

f(x) < 0 pentru orice x ∈ ℝ
