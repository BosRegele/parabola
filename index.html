unde a, b, c ∈ ℝ și a ≠ 0

Forma canonică

f(x) = a(x − m)² + n
unde V(m, n) este vârful parabolei
Abscisa vârfului
m = −b2a
Ordonata vârfului
n = −Δ4a

Completarea pătratului

ax²+bx+c
= a(x² + bax) + c
= a(x + b2a)² − b²4a + c
= a(x + b2a)² − Δ4a
Cheia întregii funcții

Discriminantul Δ

Δ = b² − 4ac
Δ > 0
Două rădăcini reale

x₁ ≠ x₂, ambele reale.
Parabola taie Ox în 2 puncte

Δ = 0
Rădăcină dublă

x₁ = x₂ = −b/(2a).
Parabola tangentă la Ox

Δ < 0
Nicio rădăcină reală

Rădăcini complexe.
Parabola nu taie Ox

Δ apare peste tot: în formula rădăcinilor x₁,₂ = (−b ± √Δ)/2a și în ordonata vârfului n = −Δ/(4a). Semnul lui decide totul.
Unde f(x) = 0

Rădăcinile ecuației

x1,2 = −b ± √Δ2a
valabilă pentru Δ ≥ 0
Δ > 0:  x1 = −b − √Δ2a , x2 = −b + √Δ2a
Δ = 0:  x1 = x2 = −b2a
Δ < 0:  ∄ soluții reale

Reconstituirea ecuației

x² − (x₁+x₂)·x + x₁·x₂ = 0

Cazuri speciale

  • b = 0 → x₁ = −x₂ (simetrice față de O)
  • c = 0 → una din rădăcini este 0
  • a = c → x₁ · x₂ = 1 (rădăcini inverse)
  • a+b+c = 0 → x₁ = 1 este rădăcină

Recapitulare
De la trei coeficienți la o curbă întreagă.
f(x) = ax² + bx + c

Discriminantul Δ = b² − 4ac decide numărul de rădăcini, semnul lui a decide direcția ramurilor, iar vârful V(−b/2a, −Δ/4a) fixează totul în plan. Asta e parabola.
